KerasInterpreter​

KerasInterpreter(source: Source, headless: Optional[bool] = False, timeout: Optional[int] = None)
Keras Interpreter device.

This device module allows to run inference of Keras models locally and is designed for test/development purposes. Output tensors are post-processed by the model post-processor function and attached to the frame.

Example:

from modlib.devices import KerasInterpreter

device = KerasInterpreter()
model = CustomKerasModel(...)
device.deploy(model)

with device as stream:
    for frame in stream:
        print(frame.detections)

ONNXInterpreter​

ONNXInterpreter(source: Source, headless: Optional[bool] = False, timeout: Optional[int] = None)
ONNX Interpreter device.

This device module allows to run inference of ONNX models locally and is designed for test/development purposes. Input and output tensors are processed by the model pre- and post-processor functions and attached to the frame.

Example:

from modlib.devices import ONNXInterpreter

device = ONNXInterpreter()
model = ONNXModel(...)
device.deploy(model)

with device as stream:
    for frame in stream:
        print(frame.detections)
