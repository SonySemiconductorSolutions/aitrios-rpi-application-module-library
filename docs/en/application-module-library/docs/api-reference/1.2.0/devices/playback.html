Playback​

Playback(recording: str, codec: Optional[FrameCodec] = <modlib.devices.playback.codecs.JsonCodec object>, headless: Optional[bool] = False, timeout: Optional[int] = None)
Playback device.

A device capable of playing back recorded frames using a specified frame codec.

Example usage:

device = Playback(recording="./path/to/my_recording.pkl", codec=PickleCodec())

with device as stream:
    for frame in stream:
        print(frame.detections)
        frame.display()

Recorder​

Recorder(directory: str, codec: FrameCodec = <modlib.devices.playback.codecs.JsonCodec object>)
Recorder class to record a series of frames to a file using a specified frame codec.

Example usage:

rec = Recorder(directory='./temp/recordings', codec=PickleCodec())

with device as stream:
    for frame in stream:
        rec.add(frame)  # Add the frame to the recording

Attributes​

  • codec: FrameCodec    The codec used for encoding/decoding frames

  • path: pathlib.Path    Recording file path

  • file: Union[IO, BinaryIO]    The open recording file object (automatically closed on exit)
